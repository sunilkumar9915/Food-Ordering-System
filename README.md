# 🍔 Food Ordering System

A full-stack food ordering web application where users can browse food items, add products to cart, and place orders.

## 🚀 Features

- User Registration & Login
- JWT Authentication
- Browse Food Menu
- Add to Cart
- Place Orders
- Admin Panel
- Order Management

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Security
- JWT Authentication
- bcrypt Password Encryption
